# MacDNS

Native DNS security for macOS. Block ads, malware, and trackers at the DNS level with DNSSEC validation.

## Download

**[Download Latest Release](https://github.com/MacDNS-app/macdns/releases/latest)**

| Requirement | Supported |
|-------------|-----------|
| macOS | 13.0+ (Ventura) |
| Architecture | Apple Silicon & Intel |

## Features

- **DNSSEC Validation** — Cryptographic verification of DNS responses
- **6 Blocklist Categories** — Ads, trackers, malware, phishing, and more
- **Custom DNS Rules** — DNS overrides and forward zones
- **Real-time Statistics** — See what's being blocked
- **Captive Portal Detection** — Works on hotel/airport WiFi
- **Zero Data Collection** — All processing happens locally on your Mac

## Installation

1. Download the `.dmg` from [Releases](https://github.com/MacDNS-app/macdns/releases/latest)
2. Drag MacDNS to Applications
3. Launch and grant permissions when prompted
4. Enable protection from the menu bar
